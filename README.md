### 코드 설명
#### 24비트 BMP 파일에서 메시지 숨기기(스테가노그래피) 
• 각 바이트의 최하위 비트(LSB)에 메시지의 1비트를 숨김 <br>
    => 따라서, 8바이트를 사용하여 메시지의 한 문자를 저장 <br> 
• "메시지 길이"(1바이트)도 BMP 파일에 인코딩하여 숨김<br><br>

#### 단계<br>
1. BMP 파일과 해당 픽셀 데이터를 읽는다.<br>
2. 각 바이트의 최하위 비트(LSB)에 메세지의 길이 정보와 메세지를 숨긴다.<br>
3. 수정된 BMP 파일을 저장합니다.<br><br><br>
